## Installation | 安装

### Step 1: Clone this repo | 克隆仓库

```bash
git clone https://github.com/AlexYoung99/wanghuiwen-product-research-skill.git
```

### Step 2: Copy the skill file into your Claude Code skills directory | 复制 Skill 文件

```bash
# Copy to your project's .claude/skills/ directory
mkdir -p .claude/skills
cp wanghuiwen-product-research-skill/wanghuiwen-product-research.md .claude/skills/
```

Or copy to your global Claude Code skills directory for all projects:

```bash
mkdir -p ~/.claude/skills
cp wanghuiwen-product-research-skill/wanghuiwen-product-research.md ~/.claude/skills/
```

### Step 3: Use it | 使用

In Claude Code, simply describe what you want to research:

在 Claude Code 中，直接描述你想调研的内容：

```
> Research the AI coding assistant market using the Wang Huiwen framework
> 用王慧文方法论帮我调研 AI 编程助手赛道
> Analyze the short video market opportunity
> 帮我分析社区团购的市场机会
```
